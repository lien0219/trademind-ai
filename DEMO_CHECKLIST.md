# TradeMind 演示验收清单（Demo Checklist）

> 用于本地 / 预发环境快速验收核心 MVP 能力。按模块勾选。

## 淘宝/天猫采集器（Beta，2026-06-05）

- [ ] 采集中心卡片显示 **淘宝/天猫采集器**，状态 **测试中**
- [ ] 批量采集按钮 **禁用**，提示「批量采集暂未开放」
- [ ] `item.taobao.com` 链接识别为 `taobao_tmall`（非 custom）
- [ ] `detail.tmall.com` / `detail.tmall.hk` 链接识别为 `taobao_tmall`
- [ ] 未登录时任务失败或提示 **LOGIN_REQUIRED**（小白可读）
- [ ] 登录后可采集 **标题、价格、至少 1 张主图**
- [ ] 详情图可采集 **至少部分**（空时不阻断草稿，可有 DETAIL_IMAGES_INCOMPLETE 提示）
- [ ] SKU 不完整时草稿可创建，提示 **SKU_INCOMPLETE**
- [ ] 价格缺失时草稿可创建，提示 **PRICE_NOT_FOUND**，发布前检查拦截
- [ ] 主图为空时任务失败 **MAIN_IMAGES_EMPTY**
- [ ] 采集成功创建商品草稿：`source=taobao_tmall`、`currency=CNY`、`status=draft`
- [ ] 商品详情页顶部显示淘宝/天猫来源提示
- [ ] 失败任务中心展示平台 `taobao_tmall`、失败原因、打开链接、打开采集浏览器、重试、生成告警
- [ ] 设置 → 采集服务 → 淘宝/天猫专属配置（登录检测、打开浏览器、超时、访问检测、重试）
