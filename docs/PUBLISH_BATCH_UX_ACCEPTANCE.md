# 批量刊登 UX 验收（Phase A2.1）

## 范围

页面：商品草稿列表、批量刊登向导、刊登任务（批次 Tab）、批次详情、失败任务中心。

分辨率：1920×1080、1440×900、1366×768、1280×800、1024×768。

## 验收结果

| 分辨率 | 商品列表多选 | 5 步向导 | 批次列表 | 批次详情 | 失败任务中心 | 备注 |
|--------|-------------|----------|----------|----------|--------------|------|
| 1920×1080 | 通过 | 通过 | 通过 | 通过 | 通过 | 批次状态 Tag 着色；子任务分页 20 |
| 1440×900 | 通过 | 通过 | 通过 | 通过 | 通过 | 向导 Steps 横向可滚动 |
| 1366×768 | 通过 | 通过 | 通过 | 通过 | 通过 | 矩阵表 `scroll.x`；工具栏按钮可换行 |
| 1280×800 | 通过 | 通过 | 通过 | 通过 | 通过 | 同 1366 |
| 1024×768 | 通过 | 通过 | 通过 | 通过 | 通过 | 窄屏依赖横向滚动 |
